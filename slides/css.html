CSS

O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem usada para estilizar documentos HTML e XML.

Principais funcionalidades:

  • Controle de layout
  • Manipulação de cores, fontes e espaçamentos
  • Responsividade para diferentes dispositivos

Variáveis CSS

As variáveis CSS permitem reutilizar valores em várias partes do código.

Declaração:


:root {
  --primary-color: #007d86;
  --secondary-color: #ff5722;
  --padding: 16px;
}

Uso:


button {
  background-color: var(--primary-color);
  padding: var(--padding);
}

Flexbox

Flexbox é um modelo de layout unidimensional usado para alinhar itens horizontal ou verticalmente.

Propriedades principais:

  • display: flex; - Ativa o Flexbox
  • justify-content - Alinhamento horizontal
  • align-items - Alinhamento vertical
  • flex-wrap - Permite quebra de linha

Exemplo:


.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  flex-wrap: wrap;
}

CSS Grid

Grid é um sistema de layout bidimensional usado para organizar elementos em linhas e colunas.

Propriedades principais:

  • display: grid; - Ativa o Grid
  • grid-template-columns - Define colunas
  • grid-template-rows - Define linhas
  • gap - Espaçamento entre itens

Exemplo:


.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 16px;
}

Trabalhando com Imagens

Propriedades importantes para lidar com imagens:

  • width e height - Dimensões da imagem
  • object-fit - Controle de redimensionamento
  • border-radius - Bordas arredondadas

Exemplo:


img {
  width: 100%;
  object-fit: cover;
  border-radius: 8px;
}

Trabalhando com Backgrounds

Propriedades para estilizar fundos:

  • background-color - Cor de fundo
  • background-image - Imagem de fundo
  • background-size - Tamanho da imagem
  • background-position - Posição da imagem

Exemplo:


.container {
  background-image: url('../assets/bg.jpg');
  background-size: cover;
  background-position: center;
}

CSS Responsivo

Use Media Queries para adaptar o layout a diferentes tamanhos de tela.

Exemplo:


@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
}
